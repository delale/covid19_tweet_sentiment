Linear Regression¶

In this exercise session we will see how to perform one- and multidimensional regression tasks on a given dataset. We will learn to use the python libraries scipy.stats as well as statsmodels to find regression parameters and their statistics.

Depending on the environment you are using, statsmodels can be installed (via terminal) by typing

  • pip3 install statsmodels
In [24]:
# Load all dependencies
import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
import scipy.stats as stats
import statsmodels.formula.api as sm
from patsy.builtins import * # tools for code evaluated in formulas (like those of statsmodels)
%matplotlib inline

For the sake of learning the libraries functionality, let's create some easy data samples following a linear relation with gaussian scatter on top of it. Let us first take a look at stats.linregress.

In [2]:
# set up some sample data
x = np.arange(100)
y = 0.5 * x + 2.0 + np.random.normal(0, 3, x.shape)

# linear regression fit
s, inter, r_value, p_value, std_err = stats.linregress(x, y)
line = s * x + inter

print('slope:\t', round(s, 3))
print('inter:\t', round(inter, 3))
print('r2 val:\t', round(r_value, 3))
print('p  val:\t', p_value)
print('std:\t', round(std_err, 3))

# plotting and comparison
f, (ax0, ax1) = plt.subplots(1, 2)
ax0.scatter(x, y)
ax0.plot(x, line, lw=3, color='red')

# plot the residuals
ax1.scatter(x, line-y)
ax1.plot(x, np.zeros_like(x), 'g--')

Data transformations¶

Nature likes power laws...

Often columns in the provided data are in fact correlated but the linear fitting requires to transform one or even both columns, e.g. through the log-transformation.

y = np.exp(x + np.random.normal(0, .2, x.shape))

bird_df = pd.read_csv("data/bird_data_vincze_etal_2015.csv")

# Perform the linear regression for (Wing area - Brain mass)
plt.figure()
bird_df.plot.scatter("Brain mass", "Wing area") # looks linear enough

reg = sm.ols(formula = "Q('Wing area') ~ Q('Brain mass')", data = bird_df).fit() # using Q tool from patsy.builtins API

Multiple linear regression¶

Let's now apply the same concepts to regress the best hyper-plane given three data columns. The following example performs a 2-dimensional regression (using least-squares method) to fit $z=a_{0} + a_{1}x + a_{2}y$

In [37]:
# set up some sample data
x = np.random.uniform(-5, 5, 100)
y = np.random.uniform(-5, 5, 100)
z = 0.3*x + 0.2*y + np.random.normal(0, 0.25, x.shape)
print(x.shape, y.shape, z.shape)

# collect them in a pandas dataframe
df = pd.DataFrame({'x': x,
                   'y': y,
                   'z': z})

# linear regression fit
reg = sm.ols(formula='z ~ x + y', data=df).fit()
print(reg.summary())
